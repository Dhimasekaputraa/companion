# Companion
Companion is a simple desktop companion application built with Electron that features a virtual cat character named Ginger.
This application is designed to be an interactive personal virtual companion.

### Features : 
- Virtual Pet Companion : Interact with Ginger, your virtual cat companion, through simple actions and animations.

- Interactive Calendar : A built-in calendar that displays the current date and allows users to set a personal birthday.

- Birthday Celebration :  Special birthday mode with unique visuals, messages, and a surprise gift when your birthday arrives.

### Installation
1. Download the Installer
You can obtain the latest version of the application through either of these methods:

Official Download Page: Visit `https://dhimasekaputraa.github.io/companion/` for the latest stable build.

GitHub Releases: Go to the [Releases](https://github.com/Dhimasekaputraa/companion/releases) section to find specific versions, changelogs, and assets.

2. Execute the Installer
Once the download is complete:

Locate the file you have download, then double-click the .exe file (e.g., companion-1.0.x.Setup.exe).

`Note: The installer uses Squirrel, which means the app will automatically install and launch itself immediately. You don't need to click "Next" or choose an installation folder.`

### Uninstall
If you need to remove the application:

1.Open Windows Settings.
2.Go to Apps > Installed Apps (Apps & Features).
3.Search for "Companion".
4.Click the three dots (or the app icon) and select Uninstall.
